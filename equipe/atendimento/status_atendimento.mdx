      2. **Pendente** - Este estado indica que o cliente está aguardando um atendente do departamento selecionado para iniciar o atendimento.
      3. **Abertos** - Este estado indica que um atendente já está em tratativa direta com o cliente.
      4. **Resolvido** - Este estado é atribuído quando a solicitação do cliente foi atendida e resolvida.
